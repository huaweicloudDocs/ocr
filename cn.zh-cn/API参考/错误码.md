# 错误码<a name="ocr_03_0028"></a>

## 功能说明<a name="section6144247617754"></a>

所有华为云的扩展接口发生400错误时，会有自定义错误信息返回，该节为您介绍每个错误码的含义。

## 返回体格式<a name="section2995295617754"></a>

```
STATUS CODE 400
```

```
{ 
        "error_code": "AIS.0005",
        "error_msg": "The service does not exist."
}
```
